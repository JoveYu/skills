# Personal Skills Repository

这是一个个人维护的 skill 仓库，用于集中管理可复用的技能说明、工作流和参考资料。

## 仓库说明

- 每个 skill 通常放在独立目录中
- 核心说明文件为 `SKILL.md`
- 相关参考文档、模板或资源会与 skill 一起存放

## 使用说明

将本仓库 clone 到以下任一目录即可：

| 位置 | 命令 |
| --- | --- |
| Claude 默认目录 | `git clone https://github.com/JoveYu/skills ~/.claude/skills` |
| Nanobot 工作目录 | `git clone https://github.com/JoveYu/skills ~/.nanobot/workspace/skills` |

clone 完成后，相关工具或代理即可从对应目录中发现并使用这些 skills。
